# Schlacht um Mittelerde2 Spracherkennung

1. Extrahiere alle sprachbezogenen Audio Dateien aus den .big files.

Ordnerstruktur:

```
audio
  bfme2
    sounds
    speech
  rotwk
    sounds
    speech
```

2. Run `fix.py`

This will fix all .wav files to make them playable in all common audio software.
